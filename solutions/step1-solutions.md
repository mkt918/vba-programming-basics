# STEP 1 解答・解説（教員用）

## STEP 1-A: 同値繰返し {#step1a}

### 【課題1】予想（n=7, 回数=5 の場合）
**正解**：A1〜A5 すべてに 7 が入る

### 【課題2】バグ発見
**間違っている行**：`Cells(a, 1).Value = a`  
**理由**：`a` はループ変数（1,2,3,4,5 と変化する）。仕様は「入力した数値 n を繰り返す」なので `n` を使うべき。

### 【課題3】修正後の正解コード
```vba
Cells(a, 1).Value = n
```

### よくある間違い
- `Cells(n, 1).Value = n`（行番号に n を使ってしまう）
- `Cells(a, 1).Value = 7`（n の代わりに具体的な数値を書いてしまう）

---

## STEP 1-B: 連番表示 {#step1b}

### 【課題1】予想（n=5 の場合）
**正解**：A1=1, A2=2, A3=3, A4=4, A5=5

### 【課題2】バグ発見
**間違っている行**：`Cells(1, 1).Value = a`  
**理由**：`Cells(1, 1)` は常に A1 を指す。ループのたびに上書きされ、最終的に A1 に最後の値だけ残る。

### 【課題3】修正後の正解コード
```vba
Cells(a, 1).Value = a
```

### 教育的ポイント
- ループ変数 `a` を「行番号」と「値」の両方に使う点が 1-A との違い
