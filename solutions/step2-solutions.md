# STEP 2 解答・解説（教員用）

## STEP 2-A: 偶数表示 {#step2a}

### 【課題1】予想（n=10 の場合）
**正解**：A列に 2, 4, 6, 8, 10 が順に入る（行は詰めて表示）

### 【課題2】バグ発見
**間違っている行**：`If a Mod 2 = 1 Then`  
**理由**：`a Mod 2 = 1` は奇数の条件。偶数は余りが 0 になるので `= 0` にすべき。

### 【課題3】修正後の正解コード
```vba
If a Mod 2 = 0 Then
```

### よくある間違い
- 偶数と奇数の条件（0と1）を混同する

---

## STEP 2-B: 奇数表示 {#step2b}

### 【課題1】予想（n=10 の場合）
**正解**：A列に 1, 3, 5, 7, 9 が順に入る

### 【課題2】バグ発見
**間違っている行**：`If a Mod 2 = 0 Then`  
**理由**：`a Mod 2 = 0` は偶数の条件。奇数は余りが 1 なので `= 1` にすべき。

### 【課題3】修正後の正解コード
```vba
If a Mod 2 = 1 Then
```

### 教育的ポイント
- 2-A と 2-B のバグは「逆」になっている。意図的な設計で条件式の意味の理解を深める。
