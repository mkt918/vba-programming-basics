📖 VBA 逆引きリファレンス

「やりたいこと」を入力すると、使うべき構文が見つかります

🔍

「」に一致する項目が見つかりませんでした。

別のキーワードで試してみてください。

変数を宣言したい 変数・宣言
構文
Dim 変数名 As 型
よく使う型：Integer（整数）/ Double（小数）/ String（文字列）/ Boolean（True/False）
使用例
Dim n As Integer Dim name As String
変数に値を代入したい 変数・宣言
構文
変数名 = 値
= の左辺に変数名、右辺に入れたい値を書く。
使用例
n = 10 name = "太郎"
ユーザーに数値・文字を入力させたい 入力
構文
変数名 = InputBox("メッセージ")
ダイアログボックスが表示され、入力した値を変数に格納できる。
使用例
n = InputBox("いくつまで表示しますか？")
セルに値を書き込みたい セル出力
構文
Cells(行, 列).Value = 値
行・列は数値で指定。列は A=1, B=2, C=3 …
使用例
Cells(1, 1).Value = 100 ' A1 に 100 Cells(a, 2).Value = a * 2 ' B行目に 2倍
「A1」など列名でセルを指定したい セル出力
構文
Range("セル番地").Value = 値
Cells は数値指定、Range は "A1" など文字列で指定。どちらも使える。
使用例
Range("A1").Value = "合計" Range("B2").Value = 500
セルの値を読み取りたい セル出力
構文
変数名 = Cells(行, 列).Value
セルの値を変数に取り込む。
使用例
x = Cells(1, 1).Value ' A1 の値を x に
セルの中身を消したい セル出力
構文
Cells(行, 列).ClearContents
値だけ消す。書式を含めて消すには .Clear。
使用例
Cells(1, 1).ClearContents Range("A1:B10").ClearContents
決まった回数だけ繰り返したい ループ
構文
For 変数 = 開始 To 終了 ' 繰り返す処理 Next 変数
変数が開始から終了まで 1 ずつ増えながら繰り返す。
使用例
For a = 1 To 5 Cells(a, 1).Value = a Next a
2つ飛ばしなど刻み幅を変えて繰り返したい ループ
構文
For 変数 = 開始 To 終了 Step 刻み ' 繰り返す処理 Next 変数
Step に負の数を使うと逆順にも繰り返せる。
使用例
For a = 2 To 10 Step 2 ' 2,4,6,8,10 Cells(a / 2, 1).Value = a Next a
出力行をループ変数と別に管理したい ループ
構文
Dim row As Integer row = 1 For a = 開始 To 終了 ' 条件を満たすときだけ出力 Cells(row, 1).Value = a row = row + 1 Next a
偶数・奇数だけ表示するときなど、出力行を別カウンタで管理する。
使用例
' 偶数だけ A列に詰めて表示 Dim row As Integer : row = 1 For a = 1 To n If a Mod 2 = 0 Then Cells(row, 1).Value = a row = row + 1 End If Next a
条件によって処理を変えたい（If） 条件分岐
構文
If 条件 Then ' 条件が True のとき End If
条件が True のときだけ処理を実行する。
使用例
If a > 10 Then Cells(1, 1).Value = "大きい" End If
条件を満たすときとそうでないときで処理を分けたい（If〜Else） 条件分岐
構文
If 条件 Then ' True のとき Else ' False のとき End If
Else 以降は条件が False のときに実行される。
使用例
If a Mod 2 = 0 Then Cells(rowA, 1).Value = a ' 偶数→A列 Else Cells(rowB, 2).Value = a ' 奇数→B列 End If
3つ以上の条件で分けたい（ElseIf） 条件分岐
構文
If 条件1 Then ' 条件1 が True ElseIf 条件2 Then ' 条件2 が True Else ' どれも False End If
使用例（FizzBuzz）
If a Mod 15 = 0 Then Cells(a,1).Value = "FizzBuzz" ElseIf a Mod 3 = 0 Then Cells(a,1).Value = "Fizz" ElseIf a Mod 5 = 0 Then Cells(a,1).Value = "Buzz" Else Cells(a,1).Value = a End If
割り算の余りを求めたい（Mod） 演算・計算
構文
結果 = 数値 Mod 割る数
余りが 0 → 割り切れる（偶数、3の倍数など）
余りが 1 → 割り切れない
使用例
10 Mod 3 ' → 1 9 Mod 3 ' → 0 （3の倍数） a Mod 2 = 0 ' a が偶数なら True
足す・引く・かける・割るを計算したい 演算・計算
構文
+ ' 足し算 - ' 引き算 * ' 掛け算 / ' 割り算（小数あり） \ ' 整数除算（小数切捨て）
使用例
Cells(a,2).Value = a * 2 ' 2倍 Cells(a,3).Value = a + 10 ' 10足す 7 / 2 ' → 3.5 7 \ 2 ' → 3 （整数部分のみ）
大小・等しいかを比べたい（比較演算子） 演算・計算
構文
= ' 等しい <> ' 等しくない > ' より大きい < ' より小さい >= ' 以上 <= ' 以下
使用例
If a = 5 Then ' a が 5 のとき If a >= 10 Then ' a が 10 以上のとき If a <> 0 Then ' a が 0 でないとき
文字列をつなげたい（&） 文字列
構文
文字列1 & 文字列2
変数と文字を組み合わせて表示するときに使う。
使用例
Cells(1,1).Value = "合計：" & goukei Cells(a,1).Value = a & "番"
数値と文字列を変換したい 文字列
構文
CStr(数値) ' 数値 → 文字列 CInt(文字列) ' 文字列 → 整数 CDbl(文字列) ' 文字列 → 小数
使用例
s = CStr(100) ' s = "100" n = CInt("42") ' n = 42
メッセージボックスを表示したい その他
構文
MsgBox "メッセージ"
プログラムの途中・終了後に結果を知らせるときに便利。
使用例
MsgBox "完了しました！" MsgBox "合計：" & goukei
コードにメモ・説明を書きたい（コメント） その他
構文
' ここに説明を書く
行頭または行末に ' を書くと、その後ろは実行されない。
使用例
' ループで 1〜n を出力する For a = 1 To n ' n 回繰り返す
特定のシートを操作したい その他
構文
Worksheets("シート名").Cells(行, 列).Value = 値
省略するとアクティブシートが対象になる。
使用例
Worksheets("Sheet1").Cells(1,1).Value = 100
マクロ（プロシージャ）を作りたい その他
構文
Sub マクロ名() ' ここに処理を書く End Sub
ひとまとまりの処理に名前をつけて定義する。Excel から実行できる。
使用例
Sub DisplayNumbers() Dim a As Integer For a = 1 To 5 Cells(a, 1).Value = a Next a End Sub
