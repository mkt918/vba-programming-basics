分類表示（If〜Else）

組み合わせ：If〜Else ＋ 複数列出力  ・  日本語手順 ＋ 穴埋め

ホーム › 応用課題② 分類表示

状況説明

STEP 2 の仕上げとして、偶数と奇数を別々の列に振り分けるプログラムを完成させましょう。

If〜Else を使って、条件によって処理を分岐させる方法を練習します。

プログラムの仕様

  1. ユーザーに「n（何まで調べるか）」を入力してもらう
  2. 偶数は A列（rowA 行目）に表示し、rowA を 1 増やす
  3. 奇数は B列（rowB 行目）に表示し、rowB を 1 増やす

例）n = 10 の場合

A列（偶数）B列（奇数）
21
43
65
87
109

If〜Else の構造

VBA 構文
—If 条件 Then
—    ' 条件が True のときの処理
—Else
—    ' 条件が False のときの処理
—End If

条件が True（偶数）→ A列に入れる、False（奇数）→ B列に入れる、という分岐を作ります。

課題 1　日本語で処理の流れを書こう

a = 1, 2, 3 のとき、それぞれどの列に入るか日本語で説明してみましょう。

例：「a = 2 のとき、2 Mod 2 = 0 なので偶数 → A列の rowA 行目に 2 を入れ、rowA を 1 増やす」
▶ 解答例を見る

a = 1：1 Mod 2 = 1 → 奇数 → B1 に 1 を入れ、rowB = 2 にする

a = 2：2 Mod 2 = 0 → 偶数 → A1 に 2 を入れ、rowA = 2 にする

a = 3：3 Mod 2 = 1 → 奇数 → B2 に 3 を入れ、rowB = 3 にする

…n まで繰り返す

課題 2　コードの穴埋めをしよう

薄黄の行の [　 ] に当てはまるコードを書いて完成させましょう。

VBA (Excel マクロ)Sub ClassifyNumbers
1Sub ClassifyNumbers()
2    Dim n As Integer
3    Dim a As Integer
4    Dim rowA As Integer
5    Dim rowB As Integer
6    n = InputBox("いくつまで調べますか？")
7    rowA = 1 : rowB = 1
8    For a = 1 To n
9        If a Mod 2 = [   ] Then
10            Cells(rowA, 1).Value = [   ]
11            rowA = rowA + 1
12        Else
13            Cells(rowB, 2).Value = [   ]
14            rowB = rowB + 1
15        End If
16    Next a
17End Sub
▶ 穴埋めの答えを見る

9行目：If a Mod 2 = 0 Then　← 偶数条件

10行目：Cells(rowA, 1).Value = a　← A列に a を入れる

13行目：Cells(rowB, 2).Value = a　← B列に a を入れる
