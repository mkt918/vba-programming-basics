偶数表示（If 文・Mod 演算子）

新しい概念：If 文による条件分岐 ＋ Mod 演算子で余りを求める

ホーム › STEP 2-A 偶数表示

状況説明

太郎さんは「1 〜 n の中から偶数だけを A列に取り出して表示する」プログラムを作りました。

しかし実行してみると、偶数ではなく奇数が表示されてしまいます。

グループで原因を調べて、正しいプログラムに直してみましょう。

Mod 演算子とは？

Mod は割り算の余りを求める演算子です。

式意味結果の例
4 Mod 24 ÷ 2 の余り0（偶数）
5 Mod 25 ÷ 2 の余り1（奇数）
a Mod 2 = 0a が偶数のとき True—
a Mod 2 = 1a が奇数のとき True—

プログラムの仕様

  1. ユーザーに「n（何まで調べるか）」を入力してもらう
  2. 1 〜 n の中から偶数だけを A列に上から順に表示する

例）n = 10 の場合

セル表示値（偶数のみ）
A12
A24
A36
A48
A510

バグのあるプログラム

※ 薄赤の行にバグが仕込まれています。

VBA (Excel マクロ)Sub DisplayEven
1Sub DisplayEven()
2    Dim n As Integer
3    Dim a As Integer
4    Dim row As Integer
5    n = InputBox("いくつまで調べますか？")
6    row = 1
7    For a = 1 To n
8        If a Mod 2 = 1 Then← バグあり
9            Cells(row, 1).Value = a
10            row = row + 1
11        End If
12    Next a
13End Sub

グループワーク

【課題 1】予想してみよう
n = 10 で実行したとき、A列にはどんな値が入ると思いますか？
【課題 2】バグを発見しよう
何行目が間違っていますか？　Mod 演算子の結果を踏まえて理由を説明しましょう。
ヒント：偶数を Mod 2 すると余りはいくつ？
【課題 3】修正しよう
正しいコードに書き直して、Excel で動作を確認しましょう。

解説（確認後に開く）

▶ バグの正解を見る

バグの場所：8行目  If a Mod 2 = 1 Then

理由：a Mod 2 = 1 は奇数の条件です。
偶数は 2 で割り切れるので余りが 0 になります。= 0 に直す必要があります。

8        If a Mod 2 = 0 Then
