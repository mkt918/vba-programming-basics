同値繰返し（For〜Next）

新しい概念：For〜Next の基本構造  ・  ループ変数と固定値の違い

ホーム › STEP 1-A 同値繰返し

状況説明

太郎さんは「数値（n）」と「回数（kaisu）」を入力すると、その数値を指定した回数だけ A列に縦に並べて表示するプログラムを作りました。

ところが実行してみると、思ったとおりの結果が出ません。
グループで原因を調べて、正しいプログラムに直してみましょう。

プログラムの仕様

  1. ユーザーに「数値（n）」と「回数（kaisu）」を入力してもらう
  2. A1 〜 A(kaisu) のセルに、数値 n を同じ値で繰り返し表示する

例）n = 7、kaisu = 5 の場合

セル表示値
A17
A27
A37
A47
A57

バグのあるプログラム

※ 薄赤の行にバグが仕込まれています。

VBA (Excel マクロ) Sub DisplayNumbers
1Sub DisplayNumbers()
2    Dim n As Integer
3    Dim kaisu As Integer
4    Dim a As Integer
5    n = InputBox("数値を入力してください")
6    kaisu = InputBox("回数を入力してください")
7    For a = 1 To kaisu
8        Cells(a, 1).Value = a← バグあり
9    Next a
10End Sub

グループワーク

【課題 1】予想してみよう
n = 7、kaisu = 5 で実行したとき、A1〜A5 にはどんな値が入ると思いますか？
グループで予想してから、実際に実行して確かめましょう。
【課題 2】バグを発見しよう
何行目が間違っていますか？なぜそうなるのか、理由を日本語で説明しましょう。
ヒント：a と n はそれぞれ何の変数でしょう？
【課題 3】修正しよう
正しいコードに書き直し、Excel で動作を確認しましょう。

解説（確認後に開く）

▶ バグの正解を見る

バグの場所：8行目  Cells(a, 1).Value = a

理由：a はループ変数で 1, 2, 3, 4, 5 と変化します。
仕様では「入力した数値 n を繰り返す」なので、a ではなく n を使う必要があります。

正しいコード：

8        Cells(a, 1).Value = n
