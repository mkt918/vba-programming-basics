複数列出力（A列 ＋ B列）

組み合わせ：複数処理 ＋ ループ内計算  ・  トレース表 ＋ 穴埋め

ホーム › 応用課題① 複数列出力

状況説明

STEP 1-B で学んだプログラムをさらに発展させます。

今回は「A列に連番」「B列にその2倍の値」を同時に表示するプログラムを完成させましょう。

コードの穴埋めと、処理の流れを日本語で書く練習もします。

プログラムの仕様

  1. ユーザーに「n（何行まで出力するか）」を入力してもらう
  2. A列（1列目）に 1, 2, 3, … n を順番に表示する
  3. B列（2列目）に A列の値の 2倍 を表示する

例）n = 5 の場合

A列（連番）B列（2倍）
12
24
36
48
510

課題 1　日本語で処理の流れを書こう

ループの中で何をしているか、1ステップずつ日本語で書いてみましょう。

例：「変数 a が 1 のとき、A1に 1 を、B1に 2 を入れる」
▶ 解答例を見る

① a = 1 のとき：A1 に 1 を入れ、B1 に 1×2=2 を入れる

② a = 2 のとき：A2 に 2 を入れ、B2 に 2×2=4 を入れる

③ a = 3 のとき：A3 に 3 を入れ、B3 に 3×2=6 を入れる

④ … n になるまで繰り返す

課題 3　コードの穴埋めをしよう

薄黄の行の [　　] に当てはまるコードを書いて、プログラムを完成させましょう。

VBA (Excel マクロ) Sub DisplayMultiColumn
1Sub DisplayMultiColumn()
2    Dim n As Integer
3    Dim a As Integer
4    n = InputBox("いくつまで表示しますか？")
5    For a = 1 To n
6        Cells(a, 1).Value = [      ]
7        Cells(a, 2).Value = [      ]
8    Next a
9End Sub
▶ 穴埋めの答えを見る

6行目：Cells(a, 1).Value = a　← A列に連番（a の値そのもの）

7行目：Cells(a, 2).Value = a * 2　← B列に2倍
