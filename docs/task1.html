複数列出力（A列 ＋ B列）

組み合わせ：複数処理 ＋ ループ内計算  ・  トレース表 ＋ 穴埋め

ホーム › 応用課題① 複数列出力

状況説明

STEP 1-B で学んだプログラムをさらに発展させます。

今回は「A列に連番」「B列にその2倍の値」を同時に表示するプログラムを完成させましょう。

コードの穴埋めと、処理の流れを日本語で書く練習もします。

プログラムの仕様

  1. ユーザーに「n（何行まで出力するか）」を入力してもらう
  2. A列（1列目）に 1, 2, 3, … n を順番に表示する
  3. B列（2列目）に A列の値の 2倍 を表示する

例）n = 5 の場合

A列（連番）B列（2倍）
12

36
48
510

課題 1　日本語で処理の流れを書こう

ループの中で何をしているか、1ステップずつ日本語で書いてみましょう。

例：「変数 a が 1 のとき、A1に 1 を、B1に 2 を入れる」
▶ 解答例を見る

① a = 1 のとき：A1 に 1 を入れ、B1 に 1×2=2 を入れる

② a = 2 のとき：A2 に 2 を入れ、B2 に 2×2=4 を入れる

③ a = 3 のとき：A3 に 3 を入れ、B3 に 3×2=6 を入れる

④ … n になるまで繰り返す

課題 3　コードの穴埋めをしよう

薄黄の行の [　　] に当てはまるコードを書いて、プログラムを完成させましょう。

VBA (Excel マクロ) Sub DisplayMultiColumn
1Sub DisplayMultiColumn()
2    Dim n As Integer
3    Dim a As Integer
4    n = InputBox("いくつまで表示しますか？")
5    For a = 1 To n
6        Cells(a, 1).Value = [      ]
7        Cells(a, 2).Value = [      ]
8    Next a
9End Sub
▶ 穴埋めの答えを見る

6行目：Cells(a, 1).Value = a　← A列に連番（a の値そのもの）

7行目：Cells(a, 2).Value = a * 2　← B列に2倍

プラス課題　2倍〜9倍を全列に出力しよう

基本課題では A列に連番・B列に2倍を出力しました。
プラス課題では C列に3倍・D列に4倍・…・I列に9倍 まで一気に出力しましょう！

例）n = 3 の場合（A〜I列）

【課題】コードの穴埋めをしよう

薄黄の行の [　　] を埋めて完成させましょう。
ヒント：列番号は A=1, B=2, C=3 … I=9 です。

VBA (Excel マクロ) Sub TimesTable
1Sub TimesTable()
2    Dim n As Integer
3    Dim a As Integer
4    n = InputBox("いくつまで表示しますか？")
5    For a = 1 To n
6        Cells(a, 1).Value = a           ' A列：連番（1倍）
7        Cells(a, 2).Value = a * 2       ' B列：2倍
8        Cells(a, [  ]).Value = a * [  ]  ' C列：3倍
9        Cells(a, [  ]).Value = a * [  ]  ' D列：4倍
10        Cells(a, [  ]).Value = a * [  ]  ' E列：5倍
11        Cells(a, [  ]).Value = a * [  ]  ' F列：6倍
12        Cells(a, [  ]).Value = a * [  ]  ' G列：7倍
13        Cells(a, [  ]).Value = a * [  ]  ' H列：8倍
14        Cells(a, [  ]).Value = a * [  ]  ' I列：9倍
15    Next a
16End Sub
▶ 穴埋めの答えを見る

 8行目：Cells(a, 3).Value = a * 3　← C列・3倍

 9行目：Cells(a, 4).Value = a * 4　← D列・4倍

10行目：Cells(a, 5).Value = a * 5　← E列・5倍

11行目：Cells(a, 6).Value = a * 6　← F列・6倍

12行目：Cells(a, 7).Value = a * 7　← G列・7倍

13行目：Cells(a, 8).Value = a * 8　← H列・8倍

14行目：Cells(a, 9).Value = a * 9　← I列・9倍

ポイント：列番号と倍率は同じ数字になっています。
パターンに気づいたら、さらに発展として「For〜Next を入れ子にしてまとめて書く」ことも挑戦してみましょう！
